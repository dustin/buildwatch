BuildWatch watches a buildbot instance and lets you know when builds are running and when they succeed or fail.

This software was written by Dustin Sallings for personal and educational use. It may also be used for anything else you feel like doing with it.

This software uses growl and twisted and all kinds of other stuff.

The LED manipulation code was written by Amit Singh. It's great for people like me who map their caps lock to control so that it's useful. :)
